# KubeVoIP RTPengine runtime

This repo contains the Dockerfile for the RTPengine image used by KubeVoIP.

Image:

```text
ghcr.io/kubevoip/kubevoip-rtpengine
```

The platform chart pins the tested tag. Most users should install KubeVoIP from
`kubevoip/kubevoip` instead of choosing component tags directly.
